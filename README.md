# InsightIQ 🧠

InsightIQ is an ultra-premium, AI-driven cognitive diagnostic platform designed to analyze neural performance through a series of specialized interactive tests. It provides deep, high-fidelity psychological reports using the Google Gemini API.

## ✨ Features

- **Interactive Cognitive Suite**: Five distinct tests focusing on specific neural dimensions:
  - **Focus**: Stroop Effect Test.
  - **Memory**: Tile Matrix (Spatial Memory).
  - **Spatial Reasoning**: 3D Rotation Match.
  - **Pattern Recognition**: Grid Continuity.
  - **Inhibition Control**: Reverse Command.
  - **Working Memory**: Chimp Span Test.
- **AI Neural Analysis**: Personalized, luxuriously crafted reports in Bengali generated by `gemini-1.5-flash`.
- **Telemetry Driven**: Results are computed based on ground-truth performance data (accuracy, reaction time, max spans).
- **Global Benchmarking**: Real-time percentile rankings and cognitive archetype identification.
- **Architectural Excellence**:
  - Full-stack Express + Vite architecture.
  - Automated CI/CD via GitHub Actions.
  - Multi-threaded build optimizations.
  - Container-ready with Docker.

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- Gemini API Key from [Google AI Studio](https://aistudio.google.com/)

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example`:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Production Build

Compile the specialized CJS server and static frontend:
```bash
npm run build
npm start
```

## 🚢 Deployment (GitHub Pages)

InsightIQ is now optimized for **GitHub Pages**. To publish:

1. **GitHub Secrets**: Add `VITE_OPENROUTER_API_KEY` to your repository's **Secrets and variables > Actions**.
2. **Push to Main**: The automated workflow will build and deploy your site.

## ⚙️ Model Configuration

You can change the AI model without touching the code:
- Edit `public/model.txt` and put any OpenRouter model ID (e.g., `google/gemini-2.0-flash-001:free`, `anthropic/claude-3-haiku`).
